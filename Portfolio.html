More Information about the Bankomaten Project

This is a simple console application for an ATM. The program manages five different users, their PIN codes, and bank accounts with balances. Users can log in, view their accounts and balances, transfer money between accounts, withdraw money, and log out.

More Information about the Numbers Game Project

The first course introduces students to the basics of programming through a series of simple console games. One of these games is a number guessing game where the player is prompted to guess a number between 1 and 20. By interacting with the game, students gain practical experience using fundamental programming concepts such as variables, conditional statements, and user input. This assignment aims to build a solid foundation for understanding more complex concepts and algorithms that will come later in the course. By making the lessons interactive and fun, students' interest and engagement in the subject are stimulated, which is crucial for their progress and learning.

More Information about Database Project

In the Database Development and Administration course the final project involves developing a console-based application for a fictional school, implementing various functionalities requested by the school administration. This includes managing personnel information, student records, course data, and salary calculations for different departments. The application utilizes both SQL queries in SQL Server Management Studio (SSMS) and Entity Framework (EF) in Visual Studio to interact with the database. The goal is to create a user-friendly and stable application that efficiently handles the school's data management needs.
